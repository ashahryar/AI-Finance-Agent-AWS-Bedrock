# Test Prompts Used During Agent Testing

## Round 1 — Basic Analysis
Use code interpreter to read the uploaded file and calculate total spending by category.

## Round 2 — Budget Suggestions
Use code interpreter to suggest a monthly budget for each category based on 
my spending. Highlight any categories where I might be overspending.

## Round 3 — Savings Recommendations
Use code interpreter to analyze my spending and give me three specific ways 
I could save money.

## Round 4 — Cross-Session Memory Test
What savings tips did you give me last time?
